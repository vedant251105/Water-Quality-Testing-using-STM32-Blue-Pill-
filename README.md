# Water Quality Monitoring System using STM32

## 📌 Overview
This project is an embedded system designed to monitor water quality in real-time using STM32 Blue Pill.

## ⚙️ Features
- Real-time water quality monitoring
- TDS (Total Dissolved Solids) measurement
- Turbidity detection
- UART and I2C communication
- Serial data output for analysis

## 🛠️ Hardware Used
- STM32 Blue Pill
- TDS Sensor
- Turbidity Sensor

## 💻 Software & Tools
- Embedded C
- STM32CubeIDE

## 🚀 Working
The system collects data from sensors, processes it using STM32, and outputs water quality parameters via serial communication.

## 📊 Applications
- Water purification systems
- Industrial monitoring
- Environmental analysis
